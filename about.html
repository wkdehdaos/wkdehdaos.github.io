서비스 소개

배고픈지도 서비스에 대해 소개합니다.

"오늘 뭐 먹지?" 이 질문은 하루에도 여러 번 반복되는 고민입니다. 선택지는 많지만 지금 이 순간 날씨와 시간에 딱 맞는 음식을 고르기는 쉽지 않습니다. 배고픈지도 서비스는 이 고민을 해결하기 위해 만들어졌습니다.

서비스 철학

이 서비스의 핵심은 단순함입니다. 버튼 하나만 누르면 현재 위치와 날씨를 분석해 지금 상황에 가장 어울리는 음식점 3곳을 바로 추천해 드립니다. 무수히 많은 선택지를 탐색하는 것이 아니라, 지금 이 순간에 최적화된 소수의 좋은 선택을 제안하는 것을 목표로 합니다.

작동 방식

1단계: 위치 확인

브라우저의 위치 정보 API를 통해 현재 위치를 파악합니다. 위치 정보는 음식점 검색에만 사용되며 서버에 저장되지 않습니다.

2단계: 날씨 분석

Open-Meteo의 기상 데이터를 통해 현재 위치의 실시간 날씨를 확인합니다. 기온, 날씨 상태(맑음/비/눈)를 파악해 적합한 음식 유형을 분석합니다.

  • 기온 25°C 이상: 냉면, 회, 초밥 등 시원한 음식 선호
  • 기온 15°C 이하: 국밥, 찌개, 고기구이 등 따뜻한 음식 선호
  • 비·눈 날씨: 파전, 칼국수 등 비 오는 날 어울리는 음식 선호

3단계: 시간대 고려

현재 시간대에 따라 아침·점심·저녁·야식에 어울리는 음식 유형이 다릅니다. 이를 반영해 추천 가중치를 조정합니다.

4단계: 주변 음식점 검색

카카오 로컬 API를 통해 반경 1km 이내 음식점을 검색합니다. 카페나 디저트 가게는 제외하고 식사 중심 음식점만 포함합니다.

5단계: 추천 선정

날씨·시간 적합도 점수와 거리를 종합해 최적의 음식점 3곳을 선정합니다. "다시 찾기" 버튼을 누르면 이미 추천한 곳을 제외하고 새로운 음식점을 보여줍니다.

사용 기술

  • 카카오 로컬 API — 주변 음식점 검색 및 상세 정보 제공
  • Open-Meteo API — 실시간 날씨 정보 (무료, API 키 불필요)
  • OpenStreetMap / Leaflet.js — 지도 표시
  • Nominatim — 위도·경도를 지역명으로 변환
  • OpenStreetMap Overpass API — 카카오 API 실패 시 백업 음식점 데이터

개인정보 보호

이 서비스는 사용자의 개인정보를 수집하거나 저장하지 않습니다. 위치 정보는 음식점 검색 요청 시 일시적으로 사용된 후 즉시 파기됩니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.

문의 및 제휴

서비스 관련 문의, 광고 제휴, 콘텐츠 협업은 홈 화면의 제휴문의 버튼을 통해 연락해 주세요. 음식점 정보 오류나 서비스 개선 제안도 환영합니다.
