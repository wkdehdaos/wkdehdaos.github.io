
# 날씨별 음식 추천

## 개요

선택한 날씨에 따라 음식을 추천하는 간단한 웹 애플리케이션입니다. 날씨를 선택하고 추천을 확인하는 상호작용적인 요소를 갖춘 현대적이고 시각적으로 매력적인 디자인을 특징으로 합니다.

## 프로젝트 개요

*   **`index.html`**: 헤더, 날씨 선택, 음식 추천 표시를 포함한 기본 HTML 구조.
*   **`style.css`**: 반응형 및 시각적으로 매력적인 사용자 인터페이스를 위한 현대적인 스타일링. CSS 변수, 최신 색상 공간 및 깔끔한 레이아웃을 사용합니다.
*   **`main.js`**: 사용자 상호작용을 처리하고, 음식 추천을 관리하며, 음식 카드를 위한 커스텀 웹 컴포넌트를 정의하는 자바스크립트 로직.

## 구현된 기능

*   HTML 구조, CSS 스타일링 및 자바스크립트 로직의 초기 설정.
*   음식 추천을 표시하기 위한 웹 컴포넌트 생성.
*   다양한 날씨 조건에 대한 음식 추천 추가.
